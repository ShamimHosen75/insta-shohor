Frequently Asked Questions

JavaScript is a high-level interpreted programming language. The website is designed with HTML & CSS, then JavaScript is mainly used to make the website more interactive and functionally rich or browser friendly.
For example :a webpage in a browser has a button created with HTML & CSS. When you click on that button, the interaction is done as a response.That interaction response is called the interactive work of JavaScript.

Local Storage:
1. Storage size is 10 MB and many people say 5 MB.
2. Data has to be manually deleted manually.
3. Works for any tab, not for any specific tab.It can be called a Universal Storage System. Where you can access storage from the browser at any time.
Session Storage:
1. Storage size 5 MB.
2. Closing the browser or tab clears the storage.
3. Works for a specific tab and closing the tab deletes that data. For example, if you put data in a tab, and open a new tab and have the URL of the previous tab, then you will not get this data. Data will be deleted from session storage.
